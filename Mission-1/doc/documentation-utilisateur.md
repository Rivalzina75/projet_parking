# Documentation utilisateur

## 1) Créer un compte
1. Aller sur `/inscription`.
2. Renseigner nom, prénom, email, mot de passe.
3. Envoyer la demande.
4. Attendre la validation administrateur.

## 2) Se connecter
1. Aller sur `/login`.
2. Saisir email + mot de passe.
3. Après validation admin, accès au dashboard.

## 3) Faire une demande de réservation
1. Ouvrir `/utilisateur/dashboard`.
2. Cliquer sur « Faire une demande de réservation ».
3. Résultat:
   - Si place libre: attribution immédiate.
   - Sinon: ajout en file d’attente avec rang affiché.

## 4) Libérer sa place
- Depuis le dashboard, cliquer « Clôturer ma réservation ».

## 5) Changer son mot de passe
1. Aller sur `/utilisateur/profil`.
2. Saisir l’ancien mot de passe.
3. Saisir et confirmer le nouveau.

## 6) Mot de passe perdu
1. Aller sur `/reset-password`.
2. Saisir l’email.
3. Utiliser le jeton affiché pour définir un nouveau mot de passe.
